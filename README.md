# 📌 Project Overview

This project demonstrates how **Business Intelligence** can transform raw transactional sales data into meaningful business insights.

Using **Microsoft Power BI**, **Power Query**, **DAX**, and **Microsoft Excel**, this dashboard analyzes historical sales performance, monitors key business KPIs, identifies trends, and predicts future sales using **Time Series Forecasting**.

The dashboard enables business stakeholders to make informed decisions through interactive visualizations and performance metrics.

---

# 🎯 Business Problem

Organizations often struggle to answer important business questions such as:

- Which products generate maximum revenue?
- Which regions perform the best?
- Which customer segment contributes the highest profit?
- How efficient is the delivery process?
- What sales trend can be expected in the coming days?

This dashboard provides a single interactive platform to answer these questions and support strategic decision-making.
